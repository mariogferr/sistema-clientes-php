# Sistema de Gerenciamento de Clientes

Sistema desenvolvido com PHP e MySQL com funcionalidades completas de CRUD.

## 🚀 Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Edição de dados
- Exclusão com confirmação

## 🛠️ Tecnologias utilizadas

- PHP
- MySQL
- HTML
- CSS

## ▶️ Como executar

1. Instalar XAMPP
2. Colocar o projeto na pasta htdocs
3. Iniciar Apache e MySQL
4. Acessar: http://localhost/sistema-clientes/pages/listar.php
